克隆音頻非同步影片翻譯軟體

立案過程

在我們的數位實驗高中，有許多與其他國家交流的機會。我們希望將作品，例如非同步課程、社團課程等，製作成國際教育交流影片。然而，這類影片可能會面臨語言障礙的問題，因為其他國家的人可能聽不懂影片中的中文內容。 因此，我計畫開發一個軟體，可以將影片中的中文內容翻譯成16種語言，並生成對應的語音。不過，這樣的功能與 Google 翻譯相似，所以我希望能進一步提升，加入克隆影片中人物音色的功能。例如，如果這是一堂烹飪課，我希望這個軟體可以將影片中的音色複製出來，將內容翻譯成日文後，重新生成一段用影片中人物音色講日文的烹飪課影片，方便分享給日本學生。

#Github #Real-time-clone-Voice #openAI

初賽

#好緊張
區賽的地點是在桃園幼獅，正式題目跟公告的題目差不多，不同的地方是資料庫的部分多了修該留言跟刪除留言，其餘的大致一樣。我當時是使用 Adobe Dreamweaver排版，VScode 寫 PHP，但因為不熟練以及很多東西忘記，像是忘記 PHP 的後端連接、忘記 SQL 語法、忘記 XAMPP 的設定等等。總之就是一團糟。最後我只完成了首頁跟登入系統，而且登入系統還是寫死的那種，如下👇
$username = admin;
$password = 1234;
if $username == admin && $password == 1234 echo "登入成功" else echo "登入失敗";
總之呢是幸運的通過初賽了，這次區賽也讓我體驗到了四個小時的高強度比賽壓力有多大，好讓我備戰壓力更大的全國賽ヾ(≧▽≦*)o

決賽

#好緊張 #Adobe Illustrator #XAMPP #PHP #JavaScript #南港展覽館
決賽地點在台北南港展覽館。當時因疫情需要消毒和戴口罩，雖然稍有不便，但並未影響比賽心情。決賽的題目新增了LOGO設計和網頁開發流程圖，這些在比賽前已經熟練練習，算是必得的分數。此外，正式題目還包括資安相關內容，如預防 SQL 注入、權限失效等，這些也是學長提前提醒並練習過的。 全國賽與區賽最大的不同在於比賽場地為開放空間，像展覽一樣，參觀者可以觀看我們的比賽，這讓壓力增加不少。比賽分兩天，第一天熟悉環境，第二天正式比賽，時間為四小時。 這次準備充分，自信滿滿。我注意到區賽時忽略的細節，例如比賽提供的題目本，其中有重要提示和評分指標。我在前 30 分鐘完成了 LOGO 和網頁開發流程圖後，直接著手分數最高的項目：資料庫 SQL、登入登出、註冊、新增留言、修改留言、刪除留言及顯示留言。由於需要模板支持這些功能，我先用 HTML 製作簡易網站，完成核心功能後再進行 CSS 優化。 相比區賽，我在全國賽的表現進步許多，但仍有遺憾：在後台與資料庫的部分花費太多時間（BUG過多），最後調整策略，先完成CSS和其他有把握的內容。修改留言功能最終未完成，留下些許缺憾。

比賽心得

#頒獎 #開心 #開啟全新的人生道路
從比賽報名到全國賽得獎只有經過短短四個月，而在這四個月中我學習到了程式語言的邏輯、網頁運作的原理、資料庫的運用、資訊安全等。而這也是我第一次參加如此大型的比賽，在南港展覽館走路都會被採訪，但其實專注在比賽的內容後就不會被外界干擾了。這是一次有趣的經驗，這次的比賽成為我人生的轉折點，為未來的生涯規劃奠定了基礎。
